Detailed Description

Class containing methods, which are necessary in order to simulate Compton scattering of gamma quanta in Compon Camera. Details of this class are described in presentation by KR available on wiki: LINK

Track * PhysicsBase::ComptonScatter ( Track *  initTrack,
DetPlane *  plane 
)

Method for Compton scattering process.

Parameters
initTrack(*Track) - track representing incident gamma quantum
plane(*DetPlane) - plane of the detector
  1. Checks whether incident Track hits the detector plane. If not returns NULL.
  2. Finds theta and phi scattering angles.
  3. Calculates gamma energy after scattering.
  4. Calculates coordinates of the new Track, representing scattered gamma quantum.
  5. Assignes values to the scattered Track object and returns it.
