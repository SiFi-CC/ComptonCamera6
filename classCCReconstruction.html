Detailed Description

Class for image reconstruction from events simulated in class CCSimulations. This class usues simple back projection algorithm based on ComptonCone class objects. Detailed description of the algorithm can be found in presentation by KR on wiki LINK

Bool_t CCReconstruction::ReconstructImage ( Int_t  iStart,
Int_t  iStop 
)

Function for image reconstruction. Sets 2D image histogram and 1D histogram of filled pixels distribution. In the loop reconstructs requested numbers of ComptonCones and looks for their intersection with the pixels of the image plane. Details of this simple back projection algorithm can be found in presentation linked in the description of this class.

Parameters
iStart(Int_t) - number of first event for image reconstruction
iStop(Int_t) - number of last event for image reconstruction
