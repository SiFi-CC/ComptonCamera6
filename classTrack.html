Detailed Description

Class, which represents single gamma ray. The Track object consists of:

  • starting point i.e. place of origin of gamma ray,
  • leading versor i.e. direction of the gamma ray,
  • energy - gamma energy in MeV,
  • name.

◆ FindCrossPoint()

Bool_t Track::FindCrossPoint ( DetPlane *  plane,
TVector3 &  position 
)

Calculates coordinates of cross point of the Track and given plane. Function returns kTRUE when there is single cross point and kFALSE in the following cases: (1) when Track is parallel to the plane and there are no cross points, (2) when plane includes the Track and all points are common, (3) when cross point is outside of the plane (size of the plane is specified in DetPlane object).

Parameters
plane(DetPlane*) - plane wich intersects with the given Track.
position(TVector3) - coordinates of the cross point returned as reference.
