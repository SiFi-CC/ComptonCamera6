Detailed Description

Class which represents plane of the detector. The DetPlane object consists of:

  • 4 coefficients of cartesian plane equation,
  • dimensions of the detector plane in Y and Z directions,
  • name.

All dimensions and distances are in mm. Cartesian plane equation: Ax + By + Cy + D = 0, where D corresponds to the distance (along X axis) between center of the coordinate system and the detector plane.
